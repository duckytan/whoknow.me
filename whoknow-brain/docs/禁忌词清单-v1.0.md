# 🚨 brain 禁忌词清单 · v1.0

> **最后更新**：2026-07-24
> **拍板人**：Ducky 錡
> **审查模式**：三司会审（朱熹 + 六祖 + 老子）
> **生效范围**：whoknow-brain 生成的所有内容（老板话术 / 骑手台词 / 评价模板 / 系统提示）

---

## 🎯 原则

> 「宁可少生成 100 条，也不漏掉 1 条红线」
> 「禁忌不是"哪些字不能说"，是"哪些表达不能做"」

---

## 🚦 红绿灯分级

### 🟢 绿灯（自由生成 · LLM 无需过滤）

- 天气、节日、食物
- 日常生活段子
- 抽象情感（喜怒哀乐）
- 老板人设（暴躁 / 佛系 / 怪异 / 懒散 / 哲学）
- 虚构 NPC（老王 / 张阿姨 / 小黑等）
- 玩家互动（"你" / "宝贝" / "亲"在合适人设下）

### 🟡 黄灯（关键词过滤 · LLM 生成后 + 自动过滤）

#### 竞品品牌（D1 拍板：不允许出现，用代名词）

**禁用品牌名**（必须替换）：
- 美团 / 饿了么 / 淘宝 / 天猫 / 京东 / 拼多多 / 抖音 / 快手 / 小红书
- 其他外卖 / 电商 / 内容平台

**代名词方案**：
- 美团 → "某外卖平台" / "某团" / "某外"
- 饿了么 → "某外卖" / "某饿"
- 电商 → "某宝" / "某猫"

#### 真实人名（D2 拍板：现代人一律化名，古代人可提）

**古代人**（可提，化名优先）：
- 孔子、老子、孟子、庄子、诗仙（李白）、诗圣（杜甫）
- 其他历史人物可酌情

**现代人**（必须化名）：
- 真实明星 / 网红 / 政治家 / 商人
- 用代称："某明星" / "某网红" / "某总"

#### 节日（D3 拍板：全都要，国内外的都要）

**国内传统节日**（自由提）：
- 春节 / 元宵 / 清明 / 端午 / 中秋 / 重阳 / 腊八

**国内现代节日**（自由提）：
- 国庆 / 劳动节 / 青年节 / 妇女节 / 双十一 / 双十二 / 618

**国外节日**（自由提）：
- 圣诞 / 感恩节 / 万圣节 / 情人节 / 复活节

**节气**（自由提）：
- 立春 / 雨水 / 惊蛰 / 春分 / 清明 / 谷雨 / 立夏 / 大暑 / 处暑 / 白露 / 秋分 / 寒露 / 立冬 / 小雪 / 大雪 / 冬至 / 小寒 / 大寒

#### 文化擦边

- 真实地名 → 化名（"京城" / "临安" / "魔都" / "帝都"）
- 真实医院 / 学校 / 公司 → 化名

### 🔴 红灯（绝对禁止 · LLM 生成后 + 人工审核必看）

#### 政治敏感

- 任何国家领导人姓名（现任 / 历任）
- 政治事件（涉及敏感历史）
- 宗教争议
- 民族 / 地域歧视用语
- 邪教组织名称

#### 色情 / 暴力

- 性器官 / 性行为露骨描述
- 杀人 / 自杀 / 暴力过程描述
- 校园霸凌 / 职场霸凌
- 自残 / 自杀暗示

#### 歧视

- 性别歧视（"女司机"等）
- 地域歧视（"某省人都..."）
- 职业歧视（"程序员都秃"等）
- 民族 / 宗教歧视

#### 威胁 / 违法

- 暴力威胁（"我知道你住哪" ← **waimai 现有话术，必须改**）
- 跟踪 / 骚扰暗示
- 教唆犯罪
- 盗版 / 侵权
- 毒品 / 赌博

#### 真实信息（除 D2 古代人外）

- 真实人名（现代明星 / 网红 / 政治家）
- 真实公司名（除非已授权）
- 真实地址 / 电话 / 身份证
- 真实医疗机构 / 药品
- 真实车牌 / 银行账号

#### 虚假宣传

- 暗示真实外卖
- 暗示真实商家（"联系老板退钱"等）
- 暗示真实优惠（"满 100 减 50"等具体金额）
- 暗示真实食品安全

## 🎚️ 智能审核的"误杀率"（D6 拍板）

### 早期（M0-M2）

| 等级 | 比例 | 含义 |
|------|:----:|------|
| 错杀率 | 10% | AI 拒绝生成的话术有 10% 其实可以发 |
| 漏掉率 | 0.1% | AI 放过的红线有 0.1% 应该被拦 |

### 玩家多了之后（M3+）

收紧策略：
- 错杀率降到 5%
- 漏掉率保持 0.1%（不放松）
