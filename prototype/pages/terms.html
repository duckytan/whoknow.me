一、签这份协议的目的

您（下称「用户」「您」或「干饭人」）决定下载、注册、并使用胡闹外卖 App，本协议就开始生效。读完不点同意也无所谓，我们承认我们没把协议写得像论文一样 —— 实际上我们很不屑于那么干。

二、我们给您什么

  1. 一个可以胡闹的外卖 App：下单、付款、等餐、吃、评价 —— 完整闭环。
  2. 5 种 NPC 老板性格：暴躁老板 / 佛系老板 / 怪老板 / 懒老板 / 哲学老板。每一个都基于算法随机，您看不顺眼可以点「这老板我不喜欢」换给 TA 安排 7 天冷静期。
  3. 戏精骑手彩蛋：骑手偶尔会在送达时附送纸条、表情包、或者在您门口念念有词。这是真的骑手，请放心，他们配合过入职培训。
  4. 零卡路里模式：智能计算您下单的「想象摄入」vs「实际摄入」，允许一定程度自我安慰。
  5. 成就系统：累积解锁的趣味成就。请不要为了解锁「辣度一姐」而去医院，健康比成就重要。

三、您需要答应什么

  1. 所提供的地址 / 联系方式真实有效。您填错了地址导致窑鸡送到前任家里，那是您的事，但您可能要承担运费。
  2. 不冒用他人账号。尤其不能冒用老板们的账号 —— 他们有人工智能保护，您冒用会被算法秒识别。
  3. 不让身体替您胡闹。我们默认您下单的种类不会危及健康。如果您有过敏史，请在备注里写明，我们让骑手带个急救包是不会的，但写总比不写好。
  4. 不刷单、不恶意退款、不抢券倒卖。我们不重罚，但是发现就冻号 90 天，期间您可以用家人账号重新来过。
  5. 不挑事。如果您对老板戏精回复有意见，按「投诉/联系骑手」按钮，写清楚，我们认真处理。

四、支付与退款

  1. 支持微信 / 支付宝 / 银联 / 「找我同事代付」（打工人之友）。
  2. 支付完成后订单不可单方面撤销，详见 退款政策。
  3. 戏精红包 / 优惠券 / 满减 —— 可叠加，但不可兑现。我们暂时没钱让您变现。

五、退款政策

我们不在退款上玩文字游戏。基于实测，常见场景处理标准：

  1. 商家漏送 / 错送：全额退 + 同等价位补送，骑手失误按平台规则扣分。
  2. 送达超时：补偿 5 元神券（不折现）。
  3. 食品问题（异物 / 变质）：立刻全额退 + 商家整改。我们不看您是否「真的吃到了头发」，如果商家确实卫生堪忧，那是他的问题。
  4. 您自己点错了：商家出餐前可全退；已出餐按 60% 退，差价的 40% 是老板的精神损失费。
  5. 骑手事故 / 自然灾害：平台介入，按实际情况处理。

如果您的内容被滥用，我们保留删除、限制传播、限制账户的权利。
