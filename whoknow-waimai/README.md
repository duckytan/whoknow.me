# 胡闹外卖（ChaosWaimai）

> **副标题**：外卖解瘾器 + 段子机 · 致敬胡闹厨房
> **Slogan**：零卡路里的外卖，越点越瘦
> **灵感来源**：韩国 FoodNeverComes（70 万人下过单）+ 美团 UI 复刻
> **状态**：🛠️ 设计 100% 完成，开发启动中（M1 进行中）
> **最后更新**：2026-07-21

## 🎯 产品定位（一句话）

> **披着美团外衣的虚拟外卖 App**：买家是真人下单，商家+骑手是 NPC，UI 8 分相似美团，满足「点外卖」的瘾但吃不到真饭。

## 🎯 4 大核心痛点（一切决策以此为准）

| # | 痛点 | 产品怎么解 |
|---|------|---------|
| ① | 无聊 | 打开 App → 刷 15 家戏精 NPC 店 → 看老板骑手翻车 |
| ② | 想笑 | NPC 暴躁/佛系/飙车人格 + 200 种话术组合 = 天然段子机 |
| ③ | 没时间没钱 | 打开像美团（0 学习成本）· 点几下就有反馈 · 不肝不氪 |
| ④ | 想减肥管不住手 | 同样的冲动，零卡路里出口 —— 满足「点外卖」的瘾但吃不到真饭 |

> **铁律**：任何功能先过「解决了哪个痛点？」——答不出就不做。
