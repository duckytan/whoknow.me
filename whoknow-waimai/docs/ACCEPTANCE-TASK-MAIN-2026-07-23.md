### Q：能不能用 main 自己电脑跑？

A：可以，但**优先用 CDP 远程浏览器**（避免本地环境干扰）。如果你电脑性能不够跑 CDP，桌面 Chrome 也行，但**要清缓存**。

### Q：真机测试找不到设备怎么办？

A：跳过真机项，标 `[⏸ skip]` + 备注"无设备"。但**移动浏览器模拟器必须跑**（Chrome DevTools 设备模拟）。

### Q：失败项太多怎么办？

A：**失败项是好事**——意味着 v18 有清晰目标。**别为了"显得通过率高"省略步骤**。

### Q：性能项 < 3 秒测不准？
