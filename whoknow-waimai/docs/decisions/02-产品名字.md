## 🔍 命名调研

### 来源灵感
锡哥自己想出来的，**致敬《胡闹厨房》（Overcooked）**——全球销量 1500 万+ 的合作烹饪神作。

### 为什么这个名字好
1. ✅ **致敬神级 IP**：Overcooked 在 Steam/Switch 玩家群体精准对位
2. ✅ **天然契合产品灵魂**：多人合作 + 三角色分工 + 戏精搞笑
3. ✅ **三 Tab 命名现成**：胡闹买家 / 胡闹老板 / 胡闹骑手
4. ✅ **社媒传播强**："胡闹外卖比胡闹厨房还胡闹" 自带话题
5. ✅ **合规友好**："胡闹"中性词 + "外卖"品类词 = 不踩美团/京东商标
6. ✅ **App Store 上架友好**：副标题明示"游戏"分类

### 必须避开的 3 个坑
1. ❌ **不复制 Overcooked 官方素材**（图、角色、关卡名）
2. ❌ **不要混淆京东外卖商标**（他们 2025-02 申请，含"外卖"二字）
3. ❌ **不要副标题只写"外卖"**（容易被判仿冒美团）
