## 📌 拍板结论

### 主体：个人开发者
- 没有公司主体
- 不能注册游戏版号（仅企业）
- 不能申请小程序小游戏（仅企业）
- 不能做经营性 ICP 备案（仅企业）

### 上线平台：Vercel 海外 + Serverless Functions
- 海外节点 → 免 ICP 备案
- Serverless Functions → 免后端服务器
- Supabase / Turso → 免数据库服务器

### 变现：纯广告
- Google AdSense / Carbon Ads（海外）
- 不能内购（无支付通道）
- 不能充值（合规问题）

### 合规：顶部 banner + 隐私政策 + 用户协议
- 顶部固定 banner：「🎮 本应用为虚拟游戏，所有订单/骑手/商家均为虚构」
- 隐私政策页（GDPR / 国内法规）
- 用户协议页（免责声明）
