## 🔴 致命风险 TOP 3（如果不做会死）

### 致命 #1：应用商店审核 + 商标侵权
- **风险**：UI 1:1 复刻美团 + "外卖"品类词 = **高概率被拒**
- **场景**：京东 2025-02 已申请"京东外卖"商标，你再申请 = 撞车
- **后果**：iOS/安卓**双端上不了** = 流量天花板砍 80%
- **必做**：商标查证（国家知识产权局）+ UI 加"游戏"标识 + 上 Web 端作回退

### 致命 #2：上线=0 流量（无传播路径）
- **风险**：Phase 1 规格**完全没写传播策略**
- **场景**：开发完成 → 上线 Vercel → 没人访问 → DAU < 10 → 失败
- **后果**：12-16 天工作量全部沉没
- **必做**：写"传播路径"章节（小红书 5 篇 KOC + 抖音 3 条切片 + V2EX 冷启动）

### 致命 #3：戏精钩子 1 周套路化
- **风险**：50 条固定模板 + 20 个写死 NPC = **重度用户 1 周看穿**
- **场景**：用户第 5 次下单看到"来了来了别催" → 截图都不截图了
- **后果**：分享率 < 1% = 传播链断
- **必做**：NPC 话术**变量化**（5 模板 × 5 变量 = 125 组合）+ 用户第 3 单解锁"老板记住你"

## 🔧 必须修复项（启动开发前必解决 · 5 项）

### 🔴 P0 #1：解决策 #002 vs #005 矛盾
**问题**：决策 #002 写"像素风"，决策 #005 写"美团 UI 1:1 复刻"——两决策并存 = 开发者混乱
**行动**：决策 #002 修订为「Phase 1 美团 1:1（像素风留 Phase 3 或放弃）」
**耗时**：5 分钟（1 行修订）
**验证**：grep "像素风" 在决策日志中只出现 1 处且明确"延后"

### 🔴 P0 #2：应用商店合规自查
**问题**：UI 1:1 复刻美团 = 高概率商标侵权 + 应用商店审核拒
**行动**：
- 查商标：国家知识产权局搜「胡闹外卖」「ChaosWaimai」
- 改 UI：顶部品牌区加"胡闹外卖 · 游戏"标识
- 改分类：应用商店分类选"游戏"非"美食"
- 回退方案：先 Web 端 PWA 验证，App 后做
**耗时**：1-2 小时（商标查证）+ 0.5 天（UI 改造）
**验证**：把 UI 截图给 5 个不看新闻的朋友，问"这是什么"——答案不该是"美团"

### 🔴 P0 #3：写"传播路径"章节
**问题**：当前规格只有"开发"和"上线"，**完全没写流量来源**
**行动**：在 00-Phase1-MVP范围.md 加一节：
```
## 📣 传播路径（与开发并行）
- Day 1-3：联系 5 个小红书 KOC（预算 ¥500）
- Day 5-7：拍 3 条抖音切片（NPC 老板甩话）
- Day 10：V2EX / 即刻 发冷启动帖
- Day 12 上线同步：所有渠道同步推
```
**耗时**：0.5 天（规划 + 联系 KOC）
**验证**：上线 1 周看各渠道引流比例

### 🔴 P0 #4：调整工期 12 → 14-16 天
**问题**：12 天工期**不现实**——Vant 主题改造 + SVG 动画调试 + 测试缓冲，**至少 14-16 天**
**行动**：决策 #004 加补丁「Phase 1 工期 = 14-16 天（不是 12 天）」
**耗时**：5 分钟（1 行修订）
**验证**：在 Day 8 时评估剩余工作量

### 🔴 P0 #5：onboarding 3 句话
**问题**：用户首次进入**不知道这是游戏不是真外卖**，可能困惑 30 秒后走
**行动**：加 onboarding：
- "这是游戏不是真外卖"
- "这是单设备游戏，换手机=重新开始"
- "老板甩话都是 NPC 别当真"
**耗时**：0.5 天（设计 + 实现）
**验证**：新用户首次进入完成率 ≥ 90%
