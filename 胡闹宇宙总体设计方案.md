## 1. 品牌与宇宙定位

| 项 | 内容 |
|---|---|
| 品牌 | whoknow · 胡闹宇宙（创始人 锡哥 / 錡哥） |
| 人格 | 胡闹 + *Who knows?* + 「家人们谁懂啊」。娱乐化、梗向、截图传播驱动 |
| 形态 | 个人开发者，**1 人无后端**；Vercel 海外部署 + 纯广告模式 |
| 节奏 | 里程碑制 **M0–M4**，无固定工期 |

**三铁律**（来自 BRAND + 三司会审 R4，不可妥协）：
1. **截图即胜利** —— 每个交互必须能回答「用户会截图发朋友圈吗？」
2. **用户习惯零摩擦** —— 像美团，别让玩家学新操作
3. **零负担** —— 不制造焦虑、不 push 留存压力（禁用 gacha 式等级 / 断签损失厌恶）

**禁忌词红线**（来自 `whoknow-brain/docs/禁忌词清单-v1.0.md`，红灯 0 容忍）：
- 制造焦虑（「你胖了」「再不买就没了」）、人身攻击（「钱多？」「你这工资也敢买」）
- 死亡 / 暴力 / 医闹 / 警察 / 爆炸 / 投毒 / 诈尸 等
- 真实明星 / 真实物价 / 真实品牌 点名（黄灯，须化名或泛化）

## 3. waimai v2 设计（核心）

### 3.1 设计支柱（5 + 1）
- **P1 每次点单 NPC 反应因你不同**（红线，记忆引擎支撑）
- **P2 截图即胜利**
- **P3 零负担**（无等级 XP / 无断签）
- **P4 戏精由选择触发，非贴图**
- **P5 单机也要有回来理由**（图鉴收集，非计数成就）
- **P6 前 3 秒必有爆点**（硬约束，见 §7）

### 3.2 核心循环
- **Moment-to-Moment（0–3s）**：进入任一关键屏 / 触发交互后 **3 秒内**撞见第一个戏精爆点。
- **Session（5–15min）**：点单 → 写备注 / 选玩法 → DRAMA 四阶段推演（接单 / 出餐 / 配送 / 送达）→ NPC 反应 → 截图 / 入图鉴。
- **Long-term**：图鉴收集（店铺 / 骑手 / 剧情分支）+ 隐藏线，驱动回访（**非 gacha**）。

### 3.3 戏精引擎 = DRAMA-ENGINE-V2（人生模拟器）
- `DramaState` 跨 4 阶段流动（bossMood 跨阶段传染：差评 → 出餐慢 → 骑手急 → 翻车送达）。
- `DramaBranch` 条件链（`branches[]` + 内联 `chain[]`）。
- `flags` 跨单叙事（`married_*` / `boss_fan_*` / `fate_reunion` …）。
- 详见 `whoknow-waimai/docs/specs/DRAMA-ENGINE-V2.md` + `DATA-STRUCTURE-v1` §3.6。

### 3.4 NPC 记忆引擎（脊柱）
- `localStorage UserStats`：`shopVisit` / `flags` / `affinity` / `memoryTier`（first / regular / vip）。
- 第 N 次点同店 → 不同话术分支（**P1 实体**）。

### 3.5 数据结构权威
- 四层：L1 内容配置（brain↔waimai 契约）/ L2 运行时 / L3 玩家持久化 / L4 图鉴成就。
- 详见 `whoknow-waimai/docs/specs/DATA-STRUCTURE-v1-2026-07-24.md`。

### 3.6 配置契约（brain↔waimai）
- envelope 6 字段：`meta` / `soul_layer` / `ui_meta` / `story_assets` / `forbidden_check` / `fallback`。
- `food.{boss, rider, branches}`；老板台词池含**记忆分级**（first / regular / vip）。
- 详见 `whoknow-brain/docs/api-spec.md` + `whoknow-waimai/docs/GDD-v2-2026-07-24.md` §9。
- **字段权威**：以 `DATA-STRUCTURE-v1` 为准（含 `moodDelta` 修正，见 §8 行动项 B）。

### 3.7 合规种子
- `whoknow-waimai/docs/specs/DRAMA-SEED-v1-2026-07-24.json`（7 分支，禁忌词已洗，可直接喂 MVP / brain）。

## 7. 前 3 秒爆点规范（硬约束 · P6）

- **定义**：进入任一关键屏 / 触发交互后 **3 秒内**必须撞见第一个戏精爆点。
- **各屏首爆点清单**：
  | 屏 | 首爆点 |
  |---|---|
  | 首页 | 商家 persona 标签（暴躁 / 哲学 / 懒） |
  | 店铺 | 暴躁老板气泡 |
  | 购物车 | 零卡路里模式提醒 |
  | 确认订单 | 备注预览（「老板会根据备注生成戏精回复」） |
  | 订单详情 | 雷速飞气泡 + 送达纸条彩蛋 |
  | 客服 | 暴躁老板抢麦 |
  | 我的订单 | 图鉴进度 |
  | Feed | 玩家相关评论（非陌生人墙） |
- **broken 标准**：
  - 同店第 5 单台词与第 1 单**无差异** = 记忆失效，**立刻否决**。
  - 任一关键屏 **3 秒无爆点** = 引导失败。
