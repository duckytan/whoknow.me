## 一、胡闹动态（A）

### 1.1 长什么样

导航栏新增 tab：`📰 动态`（排在 首页 和 订单 之间）

```
┌──────────────────────────────┐
│ 📰 胡闹动态                    │
│                               │
│ ── 20 分钟前 ──              │
│                               │
│ 匿名用户下单了                │
│ 🍜 胡闹拌面                   │
│                               │
│ 老板：「又是胡闹拌面？我       │
│   都做了八百遍了！」          │
│ 骑手：「路有点滑…不是外卖     │
│   是路有点滑」                │
│                               │
│ 💬 老王烧烤: "这客人有品味"   │
│ 😂 佛系大叔: "年轻人别熬夜"   │
│                               │
│ ❤️ 3  👍 2  📤 分享           │
│                               │
│ ── 1 小时前 ──              │
│ ...                           │
└──────────────────────────────┘
```

### 1.2 数据来源

**纯本地**——用**你自己的订单**生成动态：

| 字段 | 来源 |
|------|------|
| 菜品 | 订单 items |
| 老板吐槽 | timeline 里 stage=accepted 的 npcQuote |
| 骑手剧情 | timeline 里 stage=delivering 的 npcQuote |
| 时间 | createdAt |
| 点赞 | 随机 NPC 点赞（本地 mock，每次刷新不同）|

> **为什么要匿名**？因为只有 1 个用户（你自己）在玩，显示"你"很奇怪。匿名化后看起来像很多人在玩。

### 1.3 交互反馈（假社交）

每条动态底部固定的假互动：

```
互动    来源           效果
─────────────────────────────
点赞    NPC 自动点     3-6 个赞，每次刷新不同 NPC
评论    NPC 随机评论   老板/骑手/其他商家来评论（段子）
分享    用户手动       复制链接 / 截图引导
```

> 例：你点了一份胡闹拌面，3 分钟后动态里出现：
> - 老王烧烤老板评论：「这客人我认识，上周欠我 5 块钱」
> - 佛系大叔咖啡评论：「年轻人别熬夜」

这些**全部是本地 mock**，随机从预设文案池里抽，不需要后端。

### 1.4 和成就系统的关系

| 成就 | 关联 |
|------|------|
| 已有一枚「暴雨下单」相关 | 可触发动感动态 |
| 解锁新成就 → 自动生成一条动态 | 成就本身就是传播素材 |

## 二、送一份胡闹（B）

### 2.1 核心流程

```
用户 A 完成一单
    ↓
订单详情页 → 「📤 送一份胡闹」按钮
    ↓
生成一个分享链接：
  https://www.whoknow.me/waimai/share?dish=d001&remark=少辣&boss=angry
    ↓
A 把链接发给朋友 B（微信/QQ）
    ↓
B 点开链接
    ↓
B 看到：一份预设好的胡闹外卖（A 点的菜 + 老板性格）
    ↓
B 可以「直接下单」或「改改再下单」
    ↓
B 也下一单 → B 的订单生成 → B 也可以分享
```

### 2.2 分享链接结构

```typescript
// 纯 URL 参数，无需后端
// 格式：/waimai/share?d=<dishId>&s=<shopId>&r=<remark>&b=<bossPersonality>

// 例：老王烧烤的胡闹拌面，少放辣，暴躁老板
/share?d=d001&s=s001&r=少放辣&b=angry

// 可选参数（不传则随机）
d: dishId          // 菜品 ID
s: shopId          // 商家 ID
r: remark          // 备注（最长 50 字）
b: bossPersonality // 老板性格
```

### 2.3 接收方看到什么

```
┌──────────────────────────────┐
│  📤 你收到一份胡闹外卖！      │
│                               │
│  🍜 胡闹拌面                  │
│  🏪 老王烧烤 暴躁老板         │
│  💬 备注：少放辣              │
│                               │
│  你的朋友觉得你应该试试这个   │
│                               │
│  [ 我也要这样点 ]              │
│  [ 我自己搭配 ]                │
│                               │
│  ── 你也来胡闹一下 ──        │
└──────────────────────────────┘
```

### 2.4 新增路由

```typescript
{
  path: '/share',
  name: 'ShareOrder',
  component: () => import('@/pages/ShareOrder.vue'),
  meta: { title: '你收到一份胡闹外卖' },
}
```

`ShareOrder.vue` 读取 URL 参数，渲染预设订单页面。

## 三、A+B 融合点

| 融合 | 效果 |
|------|------|
| 动态里的「分享」按钮 → 送一份胡闹 | 刷动态 → 看到好笑的 → 分享给朋友 |
| 送完胡闹 → 回动态 | B 完成订单后 → B 的动态也出现在流里 |
| 成就解锁 → 自动生成动态 | 成就本身成为动态素材 |

```
A 下了一单
  → 自动生成动态（A 能看到）
  → A 觉得好笑 → 点分享 → 送一份给 B
    → B 打开链接 → 下了一单
      → B 的订单也生成动态
      → B 也能再分享
```
