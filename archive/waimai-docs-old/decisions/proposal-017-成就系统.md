## 一、成就列表（3 级 · ~20 枚）

### 🟢 普通（容易拿 · 玩 5 分钟就能解锁 1-2 个）

| 勋章 | 条件 | 戏精描述 |
|------|------|---------|
| 🥇 **初次点单** | 完成第 1 单 | 「恭喜你迈出了胡闹的第一步——胃来了，脑子没来」 |
| 🥇 **饿鬼出动** | 凌晨 0-5 点下单 | 「这个点了还在点外卖，你确定不是梦游？」 |
| 🥇 **月光族** | 累计消费超过 100 元（虚拟币）| 「100 块的假外卖——你真有钱」 |
| 🥇 **备注狂魔** | 留备注 3 次 | 「老板看了你的备注，把菜单撕了」 |
| 🥇 **回头客** | 同一家店点 3 次 | 「老板开始记住你的口味了——以及你欠的饭钱」 |
| 🥇 **雨神** | 雨天（浏览器天气 API）下单 | 「这种天气还点外卖？你是不是也想来送外卖？」 |

### 🔵 稀有（要玩一段时间）

| 勋章 | 条件 | 戏精描述 |
|------|------|---------|
| 🏆 **头号铁粉** | 同一家店点 10 次 | 「你已经是这家店的荣誉股东了——虽然你没投钱」 |
| 🏆 **暴躁克星** | 被 3 个不同老板骂过 | 「你不是在点外卖，是在收集老板的脏话」 |
| 🏆 **好评如潮** | 写 5 条评价 | 「你比美团 CEO 还认真」 |
| 🏆 **差评师** | 给 3 次 1 星 | 「你确定你点的不是心情？」 |
| 🏆 **时间管理大师** | 5 分钟之内完成从进店到下单 | 「手速比外卖小哥的摩托车还快」 |
| 🏆 **胡闹富翁** | 累计下单 20 单 | 「你已经花了（虚拟）500 块了——这够吃一顿真的了」 |
| 🏆 **常驻嘉宾** | 连续 7 天每天至少下 1 单 | 「你的胃已经搬来胡闹外卖了」 |

### 🟣 隐藏（特殊条件 · 列表可见但条件模糊提示 · 用户意外触发）

| 勋章 | 条件 | 戏精描述 |
|------|------|---------|
| 🥚 **真·胡闹** | 触发胡闹彩蛋（20% 订单） | 「恭喜，你被胡闹选中了——这运气建议去买彩票」 |
| 🥚 **社死现场** | 备注写了手机号/真实信息 | 「你认真了…这外卖永远不会来」 |
| 🥚 **老板的噩梦** | 同一道菜连点 10 次 | 「老板：又是胡闹拌面？我求你了换一个吧」 |
| 🥚 **骑手终结者** | 同一个骑手配送 5 次 | 「骑手已经把你拉黑了——但他逃不掉」 |
| 🥚 **马拉松选手** | 一次下单后所有 5 阶段全部看完（不跳动画） | 「你居然看完了骑手迷路的全过程——你是真爱」 |
| 🥚 **午夜凶铃** | 凌晨 3 点整下单 | 勋章是 gif 闪烁效果 👻 |

> **设计原则**：
> - 普通成就 = 新用户**前 3 单**内必拿 1-2 个（建立正反馈）
> - 稀有成就 = 留人玩的动力（玩 1 周 ~ 1 个月）
> - 隐藏成就 = 意外惊喜感（触发时弹窗最爽）

## 二、解锁流程（手动领取）

```
用户完成某种行为
    ↓
系统检测满足条件
    ↓
右下角出现角标红点/提醒：「🎖️ 新成就可领取」
    ↓
用户点开成就墙
    ↓
看到「未领取」状态 + 解锁条件
    ↓
点击「领取」按钮
    ↓
🎉 弹窗庆祝（动画 + 勋章图案 + 戏精描述）
    ↓
勋章亮起，计入成就墙
```

> **为什么手动领取**：
> - 自动领取 = 用户没感觉
> - 手动领取 = 用户有「我完成了」的成就感 + 截图分享的冲动

### 交互细节
- 普通 / 稀有 / 隐藏 **三段分区**
- 已解锁：彩色亮起 + 勋章名 + 戏精描述可见
- 未解锁：灰色锁头 + 条件可见
- 可领取：彩色+闪烁光效 +「领取」按钮
- 隐藏成就：**显示在列表中**（灰色 + 问号锁图标），**不显示具体条件**，只显示**一句模糊提示让用户去猜**

### 隐藏成就的「模糊提示」举例

| 成就 | 模糊提示（显示在卡片上）| 实际条件 |
|------|----------------------|---------|
| 🥚 真·胡闹 | 🧩「需要一点运气」| 触发胡闹彩蛋 |
| 🥚 社死现场 | 🤫「你暴露了什么？」| 备注写了手机号/真实信息 |
| 🥚 老板的噩梦 | 😈「同一道菜重复点…很多次」| 同一菜连点 10 次 |
| 🥚 骑手终结者 | 🏍️「你总遇到同一个骑手」| 同骑手配送 5 次 |
| 🥚 马拉松选手 | ⏱️「你有耐心看完吗？」| 所有 5 阶段不跳动画 |
| 🥚 午夜凶铃 | 🌙「选个阴间时间下单」| 凌晨 3 点整下单 |

> 提示要**有趣 + 有方向但不够精确**。用户看到提示会想「凌晨 3 点整？试试」而不是「这谁猜得到」。

## 四、解锁弹窗设计

### 触发时机
- 用户在成就墙**点击"领取"按钮**
- 或在订单详情/评价页面触发新成就时（通知红点引导去领取）

### 弹窗内容

```
┌──────────────────────────┐
│                          │
│      🎉                  │
│      成！就！解！锁！      │
│                          │
│    [勋章图标 - 大号]      │
│                          │
│    🥇 初次点单            │
│    「恭喜你迈出了胡闹     │
│     第一步——胃来了，     │
│     脑子没来」           │
│                          │
│       [ 分享截图 ]        │
│         [ 继续 ]          │
│                          │
└──────────────────────────┘
```

### 分享截图功能
- 点「分享截图」→ 生成一张带勋章图的卡片（纯图片）
- 用户长按保存到相册 → 自己发朋友圈
- 卡片文案 = 勋章名 + 戏精描述 + 小程序码/链接
- 这是**天然的传播素材**——用户发朋友圈炫耀勋章 = 帮我们拉新

## 五、数据结构

### localStorage 存储

```typescript
// 成就配置（固定 · 不存 localStorage）
interface Achievement {
  id: string
  name: string
  description: string  // 戏精描述
  tier: 'common' | 'rare' | 'hidden'
  icon: string         // emoji
  condition: {
    type: string       // 'order_total' | 'same_shop' | 'midnight' | 'review' | ...
    value: number
  }
}

// 用户成就状态（存 localStorage）
interface AchievementState {
  [achievementId: string]: {
    unlocked: boolean
    claimed: boolean
    progress: number   // 当前进度（如 5/10）
    unlockedAt?: number
    claimedAt?: number
  }
}

// storage key
STORAGE_KEYS.achievements = 'chaos_achievements'
```

### 进度追踪

每种成就类型的检测时机：

| 类型 | 检测时机 | 举例 |
|------|---------|------|
| 订单累计 | 下单后 | 初次点单、胡闹富翁 |
| 同一商家 | 下单后 | 回头客、头号铁粉 |
| 同一菜 | 下单后 | 老板的噩梦 |
| 时间条件 | 下单时 | 饿鬼出动、午夜凶铃 |
| 评价条件 | 提交评价后 | 好评如潮、差评师 |
| 胡闹彩蛋 | 彩蛋触发时 | 真·胡闹 |
| 骑手相关 | 配送时 | 骑手终结者 |
| 持续活跃 | 每天 | 常驻嘉宾 |
